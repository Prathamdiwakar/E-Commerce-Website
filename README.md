E-Commerce FullStack Application
Overview

This project is a Backend E-Commerce Application built using Java and Spring Boot.
It provides REST APIs to manage products, categories, and other e-commerce functionalities.

The project is designed to practice Spring Boot, REST APIs, JPA, pagination, and sorting concepts.

Tech Stack

Java

Spring Boot

Spring Data JPA

H2 Database

Maven

ModelMapper

REST APIs

Features

Create, update, delete categories

Product management APIs

Pagination support in APIs

Sorting support for API responses

RESTful API structure

JPA based database operations
